UPDATED DESIGN SYSTEM GUIDELINES
==========================================

STITCH DESIGN SYSTEM PROMPT (BASED ON ACTUAL IMPLEMENTATION):
"For all Adaplio screens, use this consistent design system based on the implemented landing page: Primary colors are orange (#FF6B35 or similar) for CTAs and accents, navy (#1F2937 or similar) for text and headers. Use clean, accessible typography with proper contrast ratios. Background should be light beige/cream (#F9F7F4) for warmth. Include subtle shadows and rounded corners (8px) for modern feel. All buttons should have hover/active states with smooth transitions. Orange buttons should be solid, secondary buttons outlined. Use consistent spacing (multiples of 8px). Include loading skeleton states for all data-dependent screens. Design for mobile-first responsive layout. Use friendly, professional illustrations similar to the exercise character on landing page. Include proper form validation styling with inline error messages. Add celebration micro-interactions with orange colors for positive actions. Maintain professional medical aesthetic while being approachable and motivational for patients of all ages."

==========================================
COLOR PALETTE (VALIDATED FROM LANDING PAGE)
==========================================

Primary Orange: #FF6B35 (CTAs, accents, progress bars)
Navy Text: #1F2937 (headings, body text)
Background Cream: #F9F7F4 (page backgrounds)
Success Green: #10B981 (completed states, positive actions)
Error Red: #EF4444 (error states, warnings)
Gray Text: #6B7280 (secondary text, placeholders)
White: #FFFFFF (cards, form backgrounds)

==========================================
RESPONSIVE BREAKPOINTS
==========================================

Mobile: 320px - 768px (single column, stacked elements, bottom navigation)
Tablet: 768px - 1024px (two-column layouts, side navigation)
Desktop: 1024px+ (multi-column grids, expanded layouts, side panels)

==========================================
ACCESSIBILITY REQUIREMENTS
==========================================

- WCAG 2.1 AA compliance
- Minimum 4.5:1 color contrast ratios (validated orange/navy combination)
- Large touch targets (44px minimum)
- Screen reader compatible
- Keyboard navigation support
- Focus indicators on all interactive elements (orange focus rings)
- Alternative text for all images and illustrations
- Clear error messaging
- High contrast mode support
- Friendly, accessible language appropriate for all ages
